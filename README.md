# CompileCore

编译原理相关算法的实现

## 目录

### 基础算法

- [x] prefix-infix-suffix: 前缀中缀后缀表达式转换
- [x] RE-NFA-DFA: 正则-非确定性有穷自动机-确定性有穷自动机
- [x] FIRST-FOLLOW-SELECT: first集 follow集 select集
- [x] LL1: LL1文法
- [ ] LR0-SLR1
- [ ] LR1
- [ ] SSD

### 应用项目

- [x] 四则运算: 支持()*/+-
- [ ] C语言编译器

## 安装 & 使用 & 文档

```bash
pip install compilecore
```
